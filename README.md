# NixOS Declarative Configuration

Multi-host NixOS flake with modular architecture for the Framework Laptop 13 AMD (7040 series).

## Hosts

| Host | Path | Description |
|------|------|-------------|
| `framework-stephan` | `modules/hosts/laptop/` | Framework Laptop 13 AMD, Btrfs + LUKS, GNOME + Niri |

## Architecture

```
.
├── flake.nix                          # Entry point: nixosConfigurations
├── install/                           # Reproducible install scripts
│   ├── install.sh                     # Orchestrator / quick reference
│   ├── scripts/
│   │   ├── 01-unmount.sh              # Close old LUKS containers
│   │   ├── 02-partition.sh            # GPT partition layout
│   │   ├── 03-luks.sh                 # LUKS format & open
│   │   ├── 04-btrfs.sh                # Btrfs filesystem + subvolumes
│   │   ├── 05-mount.sh                # Mount subvolumes to /mnt
│   │   ├── 06-generate-hardware.sh    # nixos-generate-config
│   │   ├── 07-copy-config.sh          # Copy flake into /mnt/etc/nixos
│   │   ├── 08-install.sh              # nixos-install --flake
│   │   └── 09-post-install.sh         # First-boot checklist
│   └── docs/
│       ├── PHASE.md                   # Current install phase
│       └── TODO.md                    # Full task tracker
├── modules/
│   ├── hosts/
│   │   └── laptop/
│   │       ├── default.nix            # Host config: bootloader, network, packages
│   │       └── hardware-configuration.nix # Generated by nixos-generate-config
│   ├── system/
│   │   └── btrfs-laptop.nix           # Btrfs subvolume mounts & options
│   ├── users/
│   │   └── stephan.nix                # User account + home-manager integration
│   └── desktop/
│       ├── gnome.nix                  # GNOME desktop environment
│       └── niri.nix                   # Niri Wayland compositor
└── home.nix -> /home/stephan/.config/home-manager/home.nix
```

## Quick Start (New Install)

Boot the NixOS installer USB, then:

### Option A: Run all steps automatically

```bash
git clone https://github.com/skf-funzt/nixos-config.git /home/nixos/nixos-config
cd /home/nixos/nixos-config/install/scripts
./run-all.sh
```

`run-all.sh` executes every numbered script in order and pauses after `06-generate-hardware.sh` so you can update UUIDs in `modules/system/btrfs-laptop.nix` before continuing.

### Option B: Run each step manually

```bash
cd /home/nixos/nixos-config/install/scripts
./01-unmount.sh
./02-partition.sh
./03-luks.sh
./04-btrfs.sh
./05-mount.sh
./06-generate-hardware.sh

# Edit modules/system/btrfs-laptop.nix with UUIDs from generated hardware config

./07-copy-config.sh
./08-install.sh
# Reboot into new system, then:
./09-post-install.sh
```

## Disk Layout

| Device | Size | Purpose |
|--------|------|---------|
| `/dev/nvme0n1p1` | 1 GiB | EFI System Partition (FAT32) |
| `/dev/nvme0n1p2` | ~3.6T | LUKS-encrypted Btrfs root |
| `/dev/nvme0n1p3` | 16 GiB | Swap |

### Btrfs Subvolumes

| Subvol | Mount | Options |
|--------|-------|---------|
| `@` | `/` | `compress=zstd:1,noatime` |
| `@home` | `/home` | `compress=zstd:1,noatime` |
| `@nix` | `/nix` | `compress=zstd:1,noatime` |
| `@log` | `/var/log` | `compress=zstd:1,noatime` |
| `@snapshots` | (optional) | snapshots root |

## Home Manager

Home Manager is configured as a **standalone flake** in a separate repository:

```bash
home-manager switch --flake ~/.config/home-manager#stephan
```

See: [github.com/skf-funzt/home-manager](https://github.com/skf-funzt/home-manager)
