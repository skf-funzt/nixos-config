# NixOS Declarative Configuration

Multi-host NixOS flake with modular architecture for the Framework Laptop 13 AMD (7040 series).

## Hosts

| Host | Path | Description |
|------|------|-------------|
| `framework-stephan` | `modules/hosts/laptop/` | Framework Laptop 13 AMD, Btrfs + LUKS, GNOME + Niri |

## Quick Start (New Install)

Boot the NixOS installer USB, then:

```bash
# Clone this repo
git clone https://github.com/skf-funzt/nixos-config.git /home/nixos/nixos-config
cd /home/nixos/nixos-config/install/scripts

# Run each step in order
./01-unmount.sh
./02-partition.sh
./03-luks.sh
./04-btrfs.sh
./05-mount.sh
./06-generate-hardware.sh
./07-copy-config.sh

# Edit modules/system/btrfs-laptop.nix with UUIDs from generated hardware config
# Then:
./08-install.sh

# Reboot into new system, then:
./09-post-install.sh
```

## Disk Layout

| Device | Size | Purpose |
|--------|------|---------|
| `/dev/nvme0n1p1` | 1 GiB | EFI System Partition (FAT32) |
| `/dev/nvme0n1p2` | ~3.6T | LUKS-encrypted Btrfs root |
| `/dev/nvme0n1p3` | 16 GiB | Swap |

### Btrfs Subvolumes

| Subvol | Mount | Options |
|--------|-------|---------|
| `@` | `/` | `compress=zstd:1,noatime` |
| `@home` | `/home` | `compress=zstd:1,noatime` |
| `@nix` | `/nix` | `compress=zstd:1,noatime` |
| `@log` | `/var/log` | `compress=zstd:1,noatime` |
| `@snapshots` | (optional) | snapshots root |

## Home Manager

Home Manager is configured as a **standalone flake** in a separate repository:

```bash
home-manager switch --flake ~/.config/home-manager#stephan
```

See: [github.com/skf-funzt/home-manager](https://github.com/skf-funzt/home-manager)
